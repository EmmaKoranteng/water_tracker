# Water Intake Tracker Odoo14 Module

My YouTube Video Demo :(https://youtu.be/ahPjyI5OSl8)

## Description
This Odoo 14 module is designed to help users track their daily water intake within the Odoo environment. Users can log the amount of water they drink each day and conveniently view a summary of their weekly water consumption. The data is seamlessly managed and stored within the Odoo database.

## Project Structure
- `__manifest__.py`: Main Odoo manifest file.
- `models/`: Directory containing Odoo models for water intake tracking.
- `views/`: Directory with XML files for Odoo views and interfaces.
- `tests/`: Directory for unit tests related to the Odoo module.
- `static/`: Directory for static assets like images or JavaScript files.

## How to Install and Use
1. Clone the project repository.
2. Copy the entire project directory to the Odoo addons folder.
3. Restart the Odoo server.
4. Install the module from the Odoo Apps menu.
5. Navigate to the Water Intake Tracker module and start tracking your water intake.
